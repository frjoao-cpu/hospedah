🚀 Como fazer o deploy do site

O deploy é feito automaticamente pelo GitHub Actions sempre que um commit é enviado para a branch main. Você também pode disparar um deploy manual a qualquer momento diretamente pelo GitHub:

  1. Acesse o repositório no GitHub:
  2. Clique na aba Actions (menu superior do repositório).
  3. Na lista à esquerda, selecione o workflow CI — Validação, Qualidade e Deploy.
  4. Clique no botão Run workflow (lado direito da tela).
  5. Confirme na branch main e clique em Run workflow.
  6. Aguarde o workflow concluir — o site estará atualizado em hospedah.tur.br.
▶ Abrir GitHub Actions
📋 Como copiar o código do GitHub para sua máquina

Para editar o código localmente e depois enviar as alterações, siga os passos abaixo:

  1. Instale o Git no seu computador (se ainda não tiver).
  2. Abra o terminal (ou Prompt de Comando) e execute:
  3. Entre na pasta do projeto: cd hospedah
  4. Faça suas alterações nos arquivos com o editor de sua preferência.
  5. Envie as mudanças de volta ao GitHub: git add .
    git commit -m "Descrição da alteração"
    git push origin main
  6. O deploy será disparado automaticamente após o git push.
